✦ Core Features

Everything you need, nothing you don't

Akasha is designed to be fast, private and extensible out of the box.

🔒

100% Private

All processing happens locally. Your conversations, notes and data never leave your device. No cloud required.

🧩

Skill system

Built-in tools (files, commands, web search) plus user-installable skills. Ask the agent to install from a URL or add files manually and reload.

🗣️

Natural Language

Talk to Akasha just like you would to a person. No commands to memorize — just plain English (and more languages soon).

⚡

Lightning Fast

Responses in milliseconds. No network round trips, no API latency. Akasha is always ready when you are.

🔄

Update check

Use akasha update check to see if a new version is available. Download from the Releases page when you're ready to upgrade.

🌗

Themeable UI

Choose between dark and light themes, or build your own. Akasha adapts to your workflow and aesthetics.

⚙ How it works

Get started in 3 steps

From download to fully configured in under 2 minutes.

①

Download & extract

Download the archive for your OS from the Releases page, extract it, then run akasha init and akasha start.

akasha init then akasha start
②

Add skills (optional)

Ask the agent in chat to install a skill from a URL, or copy skill files into the data folder and type /skills reload in the chat.

In chat: "Install the bankr skill from …"
③

Start chatting

Run akasha tui for the terminal interface, or launch the desktop app. Chat naturally; the agent uses tools and skills as needed.

akasha tui
